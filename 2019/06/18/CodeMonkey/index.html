第一步

下载Node.js

Hexo是由Node.js支持的博客框架，下载node.js是因为npm集成在node.js中，又因为npm使用的是国外的景象，所以要下载使用淘宝镜像的cnmp进行安装hexo，输入指令cnpm install -g hexo-cil

第二部

安装hexo

管理员的身份打开cmd 输入指令cnpm install -g hexo-cil 之后在你想要的那个盘里创建一个装blog的文件夹，进入文件夹之后，在cmd中输入 hexo init将博客进行初始化。完成后 输入hexo s 获得本地博客端口号
